# แผนการฝึกทำโจทย์ For loop + If-else สำหรับเป็น ML Engineer

| **ระดับ**            | **จำนวนโจทย์ที่แนะนำ**  | **เหตุผล**                                                                                     |
|----------------------|--------------------------|------------------------------------------------------------------------------------------------|
| **กลาง (Intermediate)** | 15 – 20 ข้อ             | - ฝึกคล่องการใช้ for loop + if-else กับข้อมูลจริง  
- ครอบคลุมเงื่อนไขหลากหลายแบบ  
- เริ่มคุ้นชินกับ nested loop และ data structure พื้นฐาน |
| **สูง (Advanced)**      | 20 – 30 ข้อ             | - ฝึก nested loop ซ้อนหลายชั้น  
- ฝึกอ่านเขียนไฟล์และจัดการข้อมูลขนาดใหญ่  
- ฝึกเขียนโค้ดที่ clean, efficient  
- ฝึกเขียน pipeline data processing แบบครบวงจร |
